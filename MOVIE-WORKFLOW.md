# RICHIESTA :

CREARE UNA APPLICAZIONE WEB CHE PERMETTE DI RENDERIZZARE A SCHERMO UNA LISTA DI FILM TRAMITE CHIAMATA API DA SERVER BACKEND SULLA STESSA MACCHINA

# SVOLGIMENTO: 

PARTENDO DAL PRESUPPOSTO DI AVERE GIA' SVOLTO LA PARTE DI BACKEND CREANDO UN ENTRY POINT E DEGLI ENDPOINT CHE RISPONDONO A RICHIESTE HTTP SU POSTMAN, SI PROCEDE ALLA CREAZIONE DELLA PARTE FRONTEND PER STAMPARE A SCHERMO.
INIZIALIZZIAMO UN PROGETTO CON VITE + REACT E SEGUIAMO QUESTI PUNTI :

1. PULIZIA DEL PROGETTO DA FILE CREATI DA VITE CHE NON SERVONO ✅
2. INSTALLAZIONE DEL NECESSARIO: REACT ROUTER (PER LA GESTIONE DELLE ROTTE IN REACT), AXIOS (PER EFFETTUARE LE CHIAMATE HTTP ALL'API), BOOTSTRAP (PER LA GESTIONE DELLO STILE QUALORA SERVISSE) ✅
3. ANDIAMO A CREARE UNA CARTELLA PAGES IN CUI ANDIAMO A CREARE DUE COMPONENTI: HOME (PER LA PAGINA PRINCIPALE, INDEX) E MOVIES (PER LA PAGINA DI OGNI SINGOLO FILM CON I SUOI DETTAGLI), IMPORTATO LA ROTTA NEL NOSTRO APP.JSX PER TESTARE CHE SI VEDA A SCHERMO ✅
4. CREIAMO ORA UNA CARTELLA COMPONENTS, DOVE ANDREMO A METTERE IL NOSTRO HEADER, CHE SARA' COMUNE A TUTTE LE PAGINE, ED UN COMPONENTE FOOTER, ANCHESSO COMUNE A TUTTE LE PAGINE
